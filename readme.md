					<h4><strong>Lëtz Learn &#8211; Android App</strong></h4>
<p>A playful method for Luxembourgish language learning by playing with several categories of quiz questions<br /> The use of this app is very simple. At start you find two buttons at the bottom of the home screen: the first (LEGAL and PRIVACY) refers to the reading of the general regulation data protection (RGPD, GDPR)<br /> The 2nd button (LET&#8217;S START) as its name suggests launches the app main page.<br /> On this page coexist the logo which is a France and Luxembourg combined flag because only two languages are used in this version; the name of the app and its slogan, as well as other different buttons.</p>
<p><strong>ABOUT</strong><br /> This button is a link to the business card of The app, it also holds 2 buttons: CLOSE: to close the window and TRANSLATE which allows the launch of the translation program.<br /> French-Luxembourg translation &#8211; type a french word or phrase into the lower text box and press the speaker icon.<br /> After a few seconds the French text will be translated into Luxembourgish. Bonus: after translation the text is read aloud with the onboard TTS (text to speech) system; cool for pronunciation mastering.<br /> Translator&#8217;s page menu bar:<br /> The + icon records he translated text in the database, thus creating a personal dictionary.<br /> The List icon shows the dictionary contents.<br /> The 3 vertical dots link to the usual icon of Android settings</p>
